## 從外部儲存庫使用 Actions

要在您自己的儲存庫中使用這些 actions，請使用 `owner/repo/.github/actions/action-name@branch` 格式來引用：

```yaml
- uses: owner/repo/.github/actions/action-name@branch
```

例如，使用 `clean-pr-cache` action：

```yaml
- uses: chia1104/chiastack.dev/.github/actions/clean-pr-cache@main
```

## 需求

- 必須在您的儲存庫中啟用 GitHub Actions
- 對於需要身份驗證的 actions，請確保在儲存庫 secrets 中配置了必要的 tokens
