`@chiastack/ui` 是一個 React UI 元件庫，提供可重複使用的 UI 元件。

## 安裝

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab value="npm">
    ```bash
    npm install @chiastack/ui
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add @chiastack/ui
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @chiastack/ui
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add @chiastack/ui
    ```
  </Tab>
</Tabs>

## 相依性

此套件需要 React 作為 peer dependency：

## 使用方式

```typescript
import { name } from '@chiastack/ui';

console.log(name); // "@chiastack/ui"
```
