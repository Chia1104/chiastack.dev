`@chiastack/utils` 提供了一系列實用的工具函式，涵蓋日期處理、錯誤處理、型別檢查、延遲執行等功能。

## 安裝

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab value="npm">
    ```bash
    npm install @chiastack/utils
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add @chiastack/utils
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @chiastack/utils
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add @chiastack/utils
    ```
  </Tab>
</Tabs>

## 基本用法

```typescript
import { delay, tryCatch, isNumber } from '@chiastack/utils';

// 延遲執行
await delay(1000);

// 安全的錯誤處理
const result = await tryCatch(fetch('/api/data'));

// 型別檢查
if (isNumber(value)) {
  console.log(value + 1);
}
```
