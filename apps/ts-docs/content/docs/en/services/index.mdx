`@chiastack/services` provides a series of common service integrations, including CAPTCHA verification and more.

## Installation

<Tabs items={['npm', 'pnpm', 'yarn', 'bun']}>
  <Tab value="npm">
    ```bash
    npm install @chiastack/services
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add @chiastack/services
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add @chiastack/services
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add @chiastack/services
    ```
  </Tab>
</Tabs>
